Resumo

Comecei abrindo -!DOCTYPE html-, abrir o -header- para o cabeçalho TechSolutions, utilizando a tag -h1- como título maior. Dentro dela optei por colocar a tag -nav- para localizar as informações na página, utilizando a tag -ul- e -li- para organizar os atributos. Também na -header- coloquei a "imagem da empresa" usando a tag -img-. A tag -section- usei três vezes na página para separar alguns tópicos. Utilizei o -article- para os principais serviços, por ser um dos pontos principais da página.
A tag -aside- coloquei para os contatos da empresa, utilizei o -ul- e -li- para organizar. E a tag -footer- coloquei as redes sociais da empresa e o endereço no maps, fazendo uso da tag -a- no qual é usada para links. E finalizado com @Copyright dentro do -footer-.
Na página, também foi usado as tags :
-br- para a quebra de linhas
-strong- para deixar em negrito o texto
-p- para fazer um parágrafo de texto
-h1...h4- para o tamanho do texto
